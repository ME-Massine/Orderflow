# OrderFlow

## Overview
OrderFlow is an event-driven order processing platform built with Spring Boot and RabbitMQ.

## Architecture
- Microservices architecture
- RabbitMQ topic exchange
- PostgreSQL (database per service)
- Docker Compose for local orchestration

## Services
- order-service
- inventory-service
- payment-service

## Tech Stack
- Java 17
- Spring Boot 3
- RabbitMQ
- PostgreSQL
- Docker
- Prometheus + Grafana (planned)
